# ViperQB - Enterprise Blockchain Security Platform

A unified blockchain middleware suite for secure file transfer, KYC verification, access control, and audit logging.

## Project Structure

```
ViperQB/
├── Front/          # Next.js frontend application
├── Back/           # Backend API and services
└── README.md       # This file
```

## Frontend (`/Front`)

The frontend is a Next.js application providing the user interface for the ViperQB platform.

**Key Features:**
- Marketing website with hero, features, pricing, and security sections
- Responsive dashboard with 11 security modules
- Real-time data widgets and charts
- Authentication flow
- Dark mode optimized design

**Tech Stack:**
- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui components

See [Front/README.md](Front/README.md) for detailed frontend documentation.

## Getting Started

### Prerequisites

- Node.js 18+ (for Frontend)
- Backend requirements (TBD)

### Frontend Setup

```bash
cd Front
npm install
cp .env.example .env.local
npm run dev
```

The frontend will be available at `http://localhost:3000`
