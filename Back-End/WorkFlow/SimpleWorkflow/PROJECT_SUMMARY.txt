================================================================================
                         BASIC USAGE EXAMPLE
================================================================================

Here's the exact code for your use case:

    from workflow import Workflow
    from trigger import TriggerType

    # Create a workflow
    wf = Workflow("Message Timer")

    # Add a trigger: after 60 seconds, send a message
    wf.add_trigger(
        trigger_name="5_min_message",
        trigger_type=TriggerType.TIME_BASED,
        action=lambda: print("It's been 5 mins!"),
        delay_seconds=60  # 60 seconds = 1 minute
    )

    # Start it
    wf.start()

    # Wait for completion
    wf.wait_for_completion()

That's it! No complex setup. No external dependencies. Just simple Python!

================================================================================

‚úì TIME-BASED TRIGGERS
  - Execute after a delay (seconds, minutes, hours)
  - Schedule at specific datetime
  - Example: Send message after 5 minutes

‚úì MANUAL TRIGGERS
  - Fire on demand (when you call wf.fire_trigger())
  - Full programmatic control
  - Example: User clicks button ‚Üí trigger fires

‚úì MULTIPLE TRIGGERS
  - Create many triggers in one workflow
  - Run in parallel at different times
  - Example: Step 1 (2s) ‚Üí Step 2 (4s) ‚Üí Step 3 (6s)

‚úì FLEXIBLE ACTIONS

  - Lambda functions for quick code
  - Pass parameters to actions
  - Return values from actions

‚úì BACKGROUND PROCESSING
  - Runs in separate thread

  - Your code continues running

‚úì NO DEPENDENCIES
  - Pure Python 3.7+
  - Uses only standard library
  - Zero external packages

‚úì MONITORING

Pattern 2: MULTIPLE STEPS AT DIFFERENT TIMES
  wf = Workflow("Pipeline")
  wf.add_trigger("s1", TriggerType.TIME_BASED, func1, 2)
  wf.add_trigger("s2", TriggerType.TIME_BASED, func2, 4)
  wf.add_trigger("s3", TriggerType.TIME_BASED, func3, 6)
  wf.start()
  wf.wait_for_completion()

Pattern 3: MANUAL CONTROL

  wf.add_trigger("action", TriggerType.MANUAL, func, 999)
  wf.fire_trigger("action")  # Fire when you want

  wf.add_trigger("task2", TriggerType.TIME_BASED, do_task2, 3)
  wf.start()
